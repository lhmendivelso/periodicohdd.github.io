# Periódico Escolar

Este es un proyecto de periódico escolar desarrollado con HTML, CSS y JavaScript.

## Contenido
- `index.html`: Página principal.
- `styles.css`: Estilos del sitio.
- `script.js`: Funcionalidad e interactividad.
